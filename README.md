# Nevin Chatbot – Artificial Intelligence Physics Tutor 

Nevin chatbot is an AI physics tutor designed to break down physics questions and guide students through solutions.

# Features 

- Physics Question Breakdown 
- Identify Topic 
- Step-by-step Solution 
- Formula and Definitions 
- Learn physics step by step 
- Personal AI tutor 

# Tech Stack 

- Lovable (No Code Development Platform) 
- Artificial Intelligence Language Model 
- Responsive Website
